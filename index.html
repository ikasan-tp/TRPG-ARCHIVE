SETUP GUIDE
1
Firebase Console でプロジェクトを作成します。
「Firestore Database」を有効にし、ルールを allow read, write: if true; に設定します。
2
プロジェクトの「設定→全般→マイアプリ→ウェブアプリを追加」でAPIキーを取得します。
表示された firebaseConfig の各値を下に貼り付けてください。
3
接続後、「CLOUD SHARE」ボタンでアーカイブ全体をFirestoreにアップロードします。
共有用URLを相手に送るだけで、スマホから閲覧できます。
